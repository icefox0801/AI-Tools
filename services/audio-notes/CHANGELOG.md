# Changelog

All notable changes to the Audio Notes service will be documented in this file.

## [1.0] - 2025-12-04

### Added
- Gradio web UI for transcription and AI summarization
- Recording management with upload and batch processing
- Real-time audio upload API for Live Captions integration
- Batch transcription with streaming progress updates
- Multi-backend ASR support (Whisper and Parakeet)
- GPU memory optimization with automatic model unloading
- AI-powered summarization using Ollama LLM
- Interactive chat for Q&A about transcribed content
- Automatic chat title generation from conversation
- Custom CSS styling with responsive design
- PWA manifest for mobile installation
- Health check endpoints for service monitoring
- FastAPI backend with Gradio UI mounting
