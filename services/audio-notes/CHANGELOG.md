# Changelog

All notable changes to the Audio Notes service will be documented in this file.

## [1.0] - 2025-12-04

### Added
- Gradio web UI for audio transcription and summarization
- Recording management with batch processing
- Multi-backend ASR support (Whisper, Parakeet)
- AI summarization and chat using Ollama LLM
- Real-time audio upload API for Live Captions
