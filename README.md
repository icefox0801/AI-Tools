# AI-Tools

Docker-based AI tools setup with PySpark.

## Contents

- `Dockerfile.pyspark` - PySpark Docker configuration
- `docker-compose.yaml` - Docker Compose configuration

## Getting Started

### Prerequisites

- Docker
- Docker Compose

### Usage

Build and run the containers:

```bash
docker-compose up -d
```

Stop the containers:

```bash
docker-compose down
```
