# Recordings Directory

This directory stores audio recordings from Live Captions.

## Files

- `recording_YYYYMMDD_HHMMSS.wav` - Audio recordings (16-bit PCM, 16kHz mono)
- `transcript_YYYYMMDD_HHMMSS.txt` - Saved transcripts

## Usage

1. Enable recording in Live Captions system tray
2. Stream audio - it gets recorded here
3. Use "Transcribe & Summarize" to open Audio Notes
4. Or run Audio Notes directly to browse recordings

## Notes

- ~1.92 MB per minute of audio
- Clear old recordings manually as needed
